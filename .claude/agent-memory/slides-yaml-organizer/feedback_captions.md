---
name: Caption style guidelines
description: Only add captions when content is distinguished; keep concise and not corny
type: feedback
---

Only add captions to gallery photos when the content is clearly distinguished from other items in the same gallery. Skip captions for visually similar siblings.

**Why:** The user explicitly requested "concise but descriptive and not corny" captions. Redundant captions clutter the presentation.
**How to apply:** Before writing a caption, ask: "Does this photo show something meaningfully different from the others in this gallery?" If not, omit the caption.
